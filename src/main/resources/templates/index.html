Wifi Project - HonestBank

die ehrliche Bank, geschrieben in Java mit Spring
Allgemein

Hierbei handelt es sich um das erste Java Projekt im Zuge einer Ausbildung am Wifi

Beschreibung

Eine kleine Fintech-Simulation, mit einem Public-Web-Bereich und jeweils einen Login-Bereich für Kunden(OnlineBanking) und Mitarbeiter(CRM)

  • Public: Produktinformationen, Zinsenrechner, Anmelde-Form
  • Banking: Stammdatenverwaltung, Produkteröffnung, Legitimation, Kontoübersichten, Buchungstabellen, Aus- & Einzahlungen, ..
  • CRM: StammdatenVerwaltung, Eröffnungs- & Legitimationsprozesse, Mitarbeiterverwaltung, Antrag- & Kontobearbeitungen
Stack

Java, Spring Boot 2, Spring MVC, Spring Security, Thymeleaf, Spring Data JPA, Hibernate, Spring Profiles(H2/Mysql)

Also: HTML, CSS(Bootstrap), Lombok, JSR-303(Validations), JaSypt(Password-Encryption for DB & Properties)

Last but not Least: a very small amount of tests using JUnit5, git, github, SSL-Certificates by LetsEncrypt
